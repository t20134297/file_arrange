Python社区拥有大量优秀的数据处理工具包，它们将科研工作中的常用的数据分析方法封装成简单易懂的函数调用，从而提高了科研工作者的数据处理效率，为科研创新提供了基础。
Numpy是高性能科学计算和数据分析的基础工具包，对繁杂的矩阵运算和操作做了极大的简化。Numpy涵盖了几乎所有常用的标准数学函数、高性能矩阵运算、常用的数据文件操作等内容。同时由于其强大的基础功能，Numpy经常被其它数据处理工具包所依赖。
Matplotlib是Python数据处理中常用的绘图库，基于Matplotlib可以使用少量代码便可以生成折线图、直方图、功率谱、条形图、错误图、散点图等。Matplotlib还提供强大的3D绘图功能和高度可配置性，使科研工作者可以很方便地绘制出高质量图片。
Scipy是常用的数据分析工具包，可以处理插值、积分、优化、图像处理、常微分方程数值解的求解等问题。Scipy提供了大量的统计学性能评估方法，可以帮助科研工作者从统计学角度快速评估模型的优略。同时提供曲线拟合相关方法，将曲线拟合的复杂过程简化。
Keras是一个基于Python编写的神经网络库，它极大简化了Tensorflow的使用门槛，简化了神经网络复杂的算法，使用者可以基于Keras快速搭建需要的神经网络层次架构并完整地配置出一个神经网络模型，并可以通过其提供的详细文档训练和使用神经网络模型。Keras为传统工程领域的科研工作者通过深度学习优化科研成果提供了很好的途径。
此外本文还结合使用了Pandas、Sklearn、SymPy、Traits和OpenCV等数据处理、图像绘制、图像处理等工具包。
